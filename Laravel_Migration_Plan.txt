MedSurvey Pro Laravel Migration Status
=====================================

Current status
--------------
The migration to Laravel is now complete at the repository-structure level.
The project root is the Laravel application, and the React frontend is served
from Laravel through Vite.

Current structure
-----------------
app/                  Laravel application code
bootstrap/            Laravel bootstrap files
config/               Laravel configuration
database/             migrations and seeders
public/               web entry point and public assets
resources/js/         React TypeScript SPA
resources/views/      Blade entry view for the SPA
routes/api.php        API routes
routes/web.php        SPA fallback route
tests/                Laravel feature tests
tests-e2e/            Playwright tests

Runtime model
-------------
- Laravel handles the API, routing, authentication middleware, models, and database.
- React remains the frontend layer and is bundled by Vite.
- `resources/views/app.blade.php` is the Laravel-rendered shell for the SPA.
- `/api/*` routes are handled by Laravel API controllers.
- Non-API browser routes fall back to the React SPA.

Useful commands
---------------
composer install
npm install
php artisan migrate --seed
npm run build
php artisan serve --host=127.0.0.1 --port=8000

Development commands
--------------------
php artisan test
npm test
npx playwright test
npm run dev

Notes
-----
This file is kept as a short migration record. Older references to a separate
`laravel-api/` folder or a separate Node/Express backend no longer describe the
current repository structure.
